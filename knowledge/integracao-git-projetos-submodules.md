# 📚 Conhecimento — Integração de Submodules Git em Projetos

## 📌 Objetivo

Definir como repositórios Git independentes são integrados dentro de `/projects` utilizando submodules.

---

## 🧩 Conceito

Submodules permitem incorporar repositórios externos dentro de um repositório principal, mantendo versionamento independente.

---

## 🔗 Comportamento Principal

- Cada projeto é um repositório Git independente
- O repositório principal armazena apenas referências de commit
- A sincronização é manual e controlada
